# Python Scripts

Questa cartella contiene tutti gli script Python del progetto:

## Script di Storia/Dati
- `hr_history.py` - Lettura storico frequenza cardiaca
- `rr_history.py` - Lettura storico intervalli RR
- `sleep.py` - Analisi dati sonno
- `sport_history.py` - Storico attività sportive
- `steps_history.py` - Storico passi
- `user_info.py` - Informazioni utente

## Script di Monitoraggio
- `realtime_monitor.py` - Monitoraggio in tempo reale
- `device_diagnostics.py` - Diagnostica dispositivo

## Utilità
- `analyze_dex.py` - Analisi file DEX Android
- `run_no_kalman.py` - Versione alternativa senza filtro Kalman

## Esecuzione

Esegui dalla root del progetto:
```bash
python run.py
```
